# vbarrai.com

Website for [vbarrai](https://github.com/vbarrai) — open-source AI tools for developers.

**Live:** https://website-vbarrai.vercel.app

## Projects

| Project | Description |
|---|---|
| [parcai](https://github.com/vbarrai/parcai) | Lightweight shell isolation for AI agents on macOS |
| [murmurai](https://github.com/vbarrai/murmurai) | Push-to-talk voice transcription and AI assistant |
| [maconfai](https://github.com/vbarrai/maconfai) | Minimal skills manager for Claude Code, Cursor and Codex |

## Stack

- **Framework:** Next.js (App Router)
- **UI:** React, Tailwind CSS
- **Language:** TypeScript
- **Testing:** Vitest + React Testing Library
- **Hosting:** Vercel

## Development

```bash
pnpm install
pnpm dev
```

## Tests

```bash
pnpm test
```
